# SoundOn Copyright Alert Bot

SoundOn Copyright Alert Bot — automated copyright claim detection, tracking, and response for BR, SPLA, and US regions.

## Overview

This project contains automation and helper scripts used to scan copyright claim inputs, track cases, send alerts, and support response workflows across regional operations.

## Notes

- Keep secrets and tokens out of the repository.
- Local OAuth/token files are ignored via `.gitignore`.
- Set runtime credentials through environment variables or secure local files that are not committed.
